# Web Scraping Toolkit – Data Extraction from Static and Dynamic Websites

A collection of 3 web scraping projects demonstrating how to handle different types of websites using Python.

## Projects

### 1. Scraping a Static Website (Books)
Extracts book data (name, price, availability) from a static website by automatically crawling 50 pages, using Requests and BeautifulSoup.
**File:** `task1_scrape_books.ipynb`

### 2. Scraping a Dynamic Website (Carrefour Egypt)
Extracts product data (name, price, link) from a real e-commerce website that loads content dynamically via JavaScript, using Selenium to control the browser and BeautifulSoup to parse the HTML.
**File:** `task2_scrape_carrefour.ipynb`

### 3. Browser Automation (YouTube Subscriptions)
Automates data extraction from a logged-in browser session's YouTube subscriptions page using Selenium, with BeautifulSoup used to extract and organize the elements.
**File:** `task3_youtube_subscriptions.ipynb`

## Skills Used
- Python
- BeautifulSoup
- Selenium
- Requests
- Web Automation
- Data Extraction
- CSV Export

## Results
Each project exports its results as a CSV file, ready for further analysis.
